PROYECTOS

Utilidades Ubuntu

Este pequeño script permite la instalación de diversas aplicaciones de uso frecuente(y no tanto) de una manera desatendida y con la ejecución de un simple comando en consola. Contiene aplicaciones como Wine(nos permite ejecutar apps de Windows), Lutris(permite jugar juegos de Windows), MegaSync, GIMP, Git, VS Code, Sublime Text entre muchas otras.

Siempre en mantenimiento...

Gestor Inventario

Esta aplicación es el trabajo práctico final del Curso Nivel 2 de Maxi Programa. Es una pequeña aplicación que permite al usuario la gestión de inventario de un comercio, en este caso de artículos de informática y tecnología.
Progamada mediante el uso de C# .Net y SQL.

Tech Tackle División Señuelos

Programa para control de producción como para control de inventario de la fabrica de señuelos Gozio. La aplicación permite la trazabilidad de los prodcutos mediante el uso de remitos y control de stock de forma automática tanto al ingreso como egreso de insumos. También permite el cálculo preciso de insumos necesarios para las tareas de emblistado, facilitando la tarea de preparado de los mismos.
Programada mediante el uso de C# .Net y SQL.
